# Automatic Theme Loader Update

Automatic theme rotation now replays the matching full-screen boot loader for every theme change.

- Matrix: binary-rain loader
- Cyan: data-network loader
- Purple: cosmic-field loader
- Ember: forge-core loader
- Red: crimson-flux loader

The portfolio remains hidden while the loader starts, preventing a flash of the next layout. The duplicate event guard ensures one loading sequence per theme change.
