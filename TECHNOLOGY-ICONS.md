# Technology Icon Upgrade

This upgrade adds animated technology symbols throughout the portfolio.

## Added

- React, Next.js, TypeScript, JavaScript, Tailwind CSS
- Flutter, Dart, Expo, Android Studio, Java
- Python, Flask, Spring Boot, Node.js
- Firebase Authentication, Cloud Firestore, Realtime Database
- MySQL, SQL, Git, GitHub, GitHub Actions
- VS Code, LaTeX, Overleaf, Vercel
- AI, debugging, documentation, API, and workflow symbols

## Main files

- `components/TechnologyBadge.tsx` — icon and color mapping
- `components/TechnologyShowcase.tsx` — animated hero technology dock
- `components/MatrixRain.tsx` — includes React, Flutter, Flask, Firebase, and coding symbols
- `app/globals.css` — badge, animation, hover, and responsive styling

Unknown technology names automatically receive a coding icon fallback, so new portfolio skills will still display correctly.
